# Grupy Facebook do monitorowania

Skill scrapuje posty z grup wypisanych niżej. **Tylko grupy publiczne** — aktor
`memo23/facebook-public-group-posts-scraper` działa bez cookies, więc do grup prywatnych
nie sięgnie (to świadomy kompromis: zero cookies do odświeżania, zero ryzyka bana konta).

Format: jedna linia = jeden URL grupy. Linie bez URL-a są ignorowane, więc można je opisywać.

- Claude Code — Polska Społeczność → https://www.facebook.com/groups/1768175597227732/
- Akademia Automatyzacji → https://www.facebook.com/groups/887930655685138/
